## Build Local

```bash
pnpm install

pnpm start:local
```

<br><br><br>

## Development

```bash
pnpm install

pnpm dev
```

<br><br><br>

## 사용법

`posts` 폴더에 블로그 게시물을 작성할 폴더를 만들고 `index.md` 파일을 작성해 주세요.

첫번째 이미지 파일이 대표 이미지로 사용됩니다.

<br><br><br>

## dark & light mode

사용자 기기의 `prefers-color-scheme`을 확인하여 자동으로 다크모드, 라이트 모드로 전환됩니다.
